Cycle8Player

Use the Cycle8Player script to load and play a JSON data file in any HTML page.

Download
‚Üì
Export JSON

Use the File > Export JSON menu item to download a JSON data file containing your image data, which can be controlled using the Cycle8Player.

Example

<html>
  <body>
    <canvas id="player"></canvas>
    <script src="./Cycle8Player.js"></script>
    <script>
      const canvas = document.getElementById("player");
      const player = new Cycle8Player(canvas);
      player.loadFromUrl("MYCYCLES.json").catch((err) => {
        console.error(err);
        alert("Failed to load MYCYCLES.json");
      });
    </script>
    <div>Lorem ipsum dolor sit amet.</div>
